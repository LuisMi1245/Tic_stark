CONSEJOS:
  • FLOAT LEFT FUNCIONA PERO NO RESPETA LA PROPIEDAD HEIGHT.
  • FLOAT SE DEFINE PARA FLOTAR IMÁGENES Y NO PARA MAQUETAR .
  • overflow: hidden; permite desaparecer el resto de imagen desbordada.
  • BOX-SIZING:border-box; encaja los elementos sin desbordar la caja padre.
  • PC-TV:
    1. Posición: {position, top, left, right, bottom, clear, z-index}
    2. Caja (modelo caja): {display, float, margin, padding, width, height}
    3. Tipografía: {font-family, font-size, line-height, color, text-align}
    4. Visual (Apartado visual): {background-color, border, border-radius, outline, opacity}

Retícula de 12 columnas y el 100% de la pantalla. Ecuación: - N-columnas * (100%/12-col) = % de N columnas A la hora de maquetar, la ecuación para elegir Col-n es: - 12 Columnas / x Número de item-container = Col-N Todos los N-item-container deben tener la clase de Col-N resultante. La suma de N-item-container debe ser igual a 12, para mantener las proporciones.
